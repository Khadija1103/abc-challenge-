# ABC Challenge

## Descripción

ABC Challenge es una aplicación web interactiva desarrollada con HTML, CSS y JavaScript que permite a los usuarios aprender y reconocer las letras del alfabeto mediante tarjetas dinámicas.

Cada tarjeta muestra una letra en la parte frontal y, al voltearla, presenta una imagen y una palabra asociada a dicha letra. El proyecto incluye efectos visuales, sonidos, filtros y un contador de progreso para hacer la experiencia más dinámica y educativa.

## Funcionalidades

* Visualización de las 26 letras del alfabeto.
* Tarjetas con efecto de volteo (flip card).
* Imagen y palabra asociada a cada letra.
* Contador de letras descubiertas.
* Filtro para mostrar únicamente las vocales.
* Botón para mostrar todas las letras.
* Botón de reinicio del juego.
* Sonido al interactuar con las tarjetas.
* Reinicio automático al completar todas las letras.
* Diseño responsive adaptable a diferentes dispositivos.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
